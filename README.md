# wight
Wight is a simple Brainfuck interpreter written in ANSI C.

## Usage
```
$ make
$ ./bin/wight <file>
```

## Naming
The name references the "Miss Peregrine's Home for Peculiar Children" series of books by Ransom Riggs.
